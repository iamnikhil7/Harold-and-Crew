# Images

Place general lifestyle / product imagery here (hero photos, onboarding
card imagery, activity cards, etc.).

Suggested naming:

- `landing-hero.jpg` — full-bleed landing background
- `onboarding/1-mornings-slow.jpg`, `onboarding/1-mornings-coffee.jpg`, …
- `activities/saturday-run.jpg`, `activities/pickup-basketball.jpg`, …

If this folder is empty, the app falls back to Unsplash URLs configured
in `next.config.ts` and `src/lib/questions.ts`.
